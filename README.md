## 📦 Installation

1. Clone the repository
```bash
git clone https://github.com/Xinoxi/Modern-Clock-for-Quickshell.git
```
2. Create Quickshell config folder if you do not have
```bash
mkdir -p ~/.config/quickshell
```

3. Copy widget files
```bash
cp -r Modern-Clock-for-Quickshell/Widget ~/.config/quickshell/
cp Modern-Clock-for-Quickshell/shell.qml ~/.config/quickshell/
```

4. Run
```bash
quickshell
```
